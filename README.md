## BalmJS template - pwa
> A PWA project

## Installation

```sh
npm install -g balm-cli
balm init pwa my-project

cd my-project
yarn
# or
npm i
```

> [manifest.json](https://app-manifest.firebaseapp.com/)

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

> thx [BalmJS](http://balmjs.com/)

# BalmJS template - `pwa`

> A PWA project

## Installation

```sh
npm install -g balm-cli
balm init pwa my-project

cd my-project
yarn
# or
npm i
```

- [App Manifest Generator](https://app-manifest.firebaseapp.com/)

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

- thx [BalmJS](https://balmjs.com/)
